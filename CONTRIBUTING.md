# Contributing

Thanks for your interest in contributing!

- **Adding a new shortcut:** see [`docs/HOW-TO-CREATE-NEW-SHORTCUT.md`](docs/HOW-TO-CREATE-NEW-SHORTCUT.md).
- **Running tests:** see [`docs/TESTING.md`](docs/TESTING.md).
- **Reporting bugs:** open an issue at [GitHub Issues](https://github.com/robinsalehjan/vscode-xcode-shortcuts/issues).

## Pull Request Checklist

- [ ] `npm test` passes locally
- [ ] `package.json` keybinding has all three platform keys (`mac`, `win`, `linux`)
- [ ] `docs/SHORTCUTS.md` table updated
- [ ] `CHANGELOG.md` entry added (date format: DD.MM.YYYY)
